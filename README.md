# Проект машинного обучения: Предсказание выживаемости на Титанике

## Описание проекта
Это мой первый полноценный проект по машинному обучению, выполненный в рамках обучения. Цель - предсказать, выживет ли пассажир на Титанике (`Survived: 0` или `1`), используя данные из соревнования Kaggle.

## Используемые инструменты
*   **Язык:** Python
*   **Библиотеки:** Pandas (для анализа и обработки данных), Scikit-learn (для моделей ML)
*   **Среда:** VS Code, Jupyter Notebook, Conda
*   **Система контроля версий:** Git, GitHub

## Ход работы и эксперименты
Я тестировал различные подходы и фиксировал результаты:

1.  **Baseline (Ручные правила IF/ELSE):** Если женщина — выжила, иначе — нет.
    *   *Результат на Kaggle:* **76.555%**
2.  **Feature Engineering (Признаки: Pclass, Sex, IsAlone, Age, Fare):** Использование алгоритма Random Forest.
    *   *Результат на Kaggle:* 74.88% (Проблема переобучения/масштабирования)
3.  **Улучшенный Feature Engineering (Добавлен признак Title) и настройка гиперпараметров:** Использование оптимальных параметров для Random Forest.
    *   *Финальный результат на Kaggle:* **77.511%** (Превысил Baseline)

## Финальный результат
Моя лучшая модель (Random Forest с настроенными параметрами и признаками Title, Sex, Pclass, Age, Fare, IsAlone) показала точность **77.511%** на тестовом наборе данных Kaggle.
